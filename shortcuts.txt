COMMON:
	space:      Show tools menu

SELECTION:
	a:          Select All / Deselect All
	c:          Close Selection (selects all associated elements of lower dimension)

SUBSETS:
	Alt + s:    Assign Subset

MARKS:
	m:          Mark Selection

REFINEMENT:
	Alt + r:    Refine
	
TOPOLOGY:
	Delete:     Erase Selected Elements
	Alt + m:    Merge
	Alt + c:    Collapse Edge
	Alt + w:    Swap Edge

COORDINATE TRANSFORM:
	g:          Grab (move)
	s:          Scale
	x:          Restrict to x axis (during grab / scale / rotate)
	y:          Restrict to y axis (during grab / scale / rotate)
	z:          Restrict to z axis (during grab / scale / rotate)

INFO:
	i:          Print Geometry Info
	Alt + i:    Print Selection Center
